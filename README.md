# לוח ניתוח שחמט

לוח שחמט לניתוח וריאציות — שימושי לבדיקת מהלכים תוך כדי משחק מקוון.

## הפעלה

```bash
npm install
npm run dev
```

פתח [http://localhost:3000](http://localhost:3000) בדפדפן.

## תכונות

- **גרירה ולחיצה** — הזזת כלים לשני הצדדים
- **הדגשת מהלכים חוקיים** — לחץ על כלי לראות לאן הוא יכול לזוז
- **ניווט במהלכים** — חצים ← → (או כפתורים) לצפייה בהיסטוריית המהלכים
- **קיצורי מקלדת** — `←` / `→` ניווט, `Home` / `End` ראשון / אחרון
- **טעינת FEN** — הדבק FEN לדמות עמדה ספציפית
- **ייצוא PGN** — העתקת רשימת המהלכים
- **כלים שנלקחו** — הצגה עם יתרון חומרה
- **סיבוב לוח** — נקודת מבט שחור/לבן
- **מצב כהה/בהיר**

## טכנולוגיות

- React 19 + TypeScript
- chess.js (לוגיקת שחמט)
- react-chessboard v5 (לוח ויזואלי)
- Tailwind CSS v4
- Vite
